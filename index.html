FIZZER A Coding Project

Classic fizz buzz in an attractice format. This is a JavaScript coding exercise using loops. Write a program that prints the numbers from 1 to 100 and for multiples of '3' display "Fizz" instead of the number and for multiples of '5' display "Buzz". If it is a multiple of both, display "FizzBuzz."
